## App Description
QuoteWise is a mobile application designed to assist construction companies 
and professionals with smart budgeting and quotation management. The app 
allows project managers and contractors to create and manage construction 
projects, record material, labour, and equipment costs, and automatically 
generate accurate client-ready quotations. With real-time budget tracking 
and secure Firebase data storage, QuoteWise eliminates the inefficiencies 
of manual cost estimation, reduces errors, and helps prevent budget overruns 
and project delays. The application is designed for Android and iOS devices, 
making it accessible and easy to use for non-technical users in the 
construction industry.
